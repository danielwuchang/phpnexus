# PHP Installation #

[PHPBrew](https://github.com/phpbrew/phpbrew)

 * phpbrew builds and installs multiple version php(s) in your $HOME directory. phpbrew also manage the environment variables, so you can use, switch php version whenever you need.

[phpfarm](https://github.com/cweiske/phpfarm)

 * Set of scripts to install a dozen of PHP versions in parallel on a system. It also creates a Pyrus installation for each PHP version. Primarily developed for PEAR's continuous integration machine.
